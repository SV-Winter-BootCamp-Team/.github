## 🚀 How to Start

Clone Repository

```
git clone https://github.com/SV-Winter-BootCamp-TeamD/backend.git
```

### .env setting in the Settings folder

frontend/.env

```jsx
VITE_BASE_URL = 
VITE_SOCKET_URL = 
```

backend/.env

```jsx
SECRET_KEY=

DATABASE_NAME=
DATABASE_USERNAME=
DATABASE_PASSWORD=
DATABASE_HOST=
DATABASE_PORT=
MYSQL_ROOT_PASSWORD=

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_STORAGE_BUCKET_NAME=
AWS_S3_REGION_NAME=

OPENAI_API_KEY=
PIXABAY_API_KEY=
```

### Run Docker

```
docker compose -f docker.compose.prod.yml up --build
```
